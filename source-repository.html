Overview

This project uses Git to manage its source code. Instructions on Git use can be found at http://git-scm.com/documentation.

Web Browser Access

The following is a link to a browsable version of the source repository:

https://github.com/nom-tam-fits/nom-tam-fits

Anonymous Access

The source can be checked out anonymously from Git with this command (See http://git-scm.com/docs/git-clone):

$ git clone git@github.com:nom-tam-fits/nom-tam-fits.git

Developer Access

Only project developers can access the Git tree via this method (See http://git-scm.com/docs/git-clone).

$ git clone git@github.com:nom-tam-fits/nom-tam-fits.git

Access from Behind a Firewall

Refer to the documentation of the SCM used for more information about access behind a firewall.
